# REST TODO App

A simple CRUD for a 'TODO app'

## API

https://todo-app-rest-api-br.herokuapp.com/


| Method |   URL   |     Action    |
|:------:|:-------:|:-------------:|
| GET    | '/<id>' |   get a task  |
| POST   |   '/'   | create a task |
| PUT    | '/<id>' | update a task |
| DELETE | '/<id>' | delete a task |
